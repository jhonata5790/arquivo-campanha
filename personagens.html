Nota de segurança

Dossiês bloqueados não indicam necessariamente ameaça, morte, traição ou envolvimento paranormal. Em alguns casos, significam apenas que a Ordem ainda não tem informação suficiente para registrar alguém com segurança.

AVISO // A ausência de informação também é uma forma de proteção.

O arquivo não mostra tudo. Ele mostra o que a Ordem permite que seja visto.

Nota interna // central de dossiês

Um dossiê nunca é só uma ficha. É uma tentativa de transformar alguém em informação. A Ordem faz isso para proteger agentes. Às vezes, faz para controlar o quanto eles sabem sobre si mesmos.
